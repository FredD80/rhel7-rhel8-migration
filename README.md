# RHEL 7 to RHEL 8 Migration Automation

This repository contains the migration specification and automation scaffold for side-by-side RHEL 7 to RHEL 8 migrations in AWS.

Primary documents:

- [End-to-end migration specification](rhel7-to-rhel8-end-to-end-migration.md)
- [Migration workflow diagram](docs/migration-workflow-diagram.md)

The playbook files are scaffolds. They define the required structure and intentionally fail until their implementation is added.
